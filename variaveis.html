palavras reservadas

" SE ENTAO SENAO FIMSE "
" PARA DE ATE PASSO PROXIMO "
" ENQUANTO FAZ FIMENQUANTO "
" REPETE ATE "
" INTEIRO REAL CARACTER LOGICO TEXTO "
" CONSTANTE "
" VERDADEIRO FALSO "

simbolos permitidos
"abcdefghijklmnopqrstuvxywz"
"ABCDEFGHIJKLMNOPQRSTUVXYWZ"
"_0123456789"

TEXTO - começa e termina com aspas ""
exemplo:
texto nome := "ANTONIO"

CARACTER - começa e termina com aspas ""
tem um e um só carecter
exemplo:
caracter sexo := "M"

LOGICO - pode ter os valores VERDADEIRO e FALSO exemplo:
logico repetente := falso

INTEIRO - pode ter os valores inteiro exemplo:
inteiro idade := 14543

REAL - pode ter os valores reais no formato xx.yy exemplo:
real media := 14.5
