Portugol

A Programação de computadores é uma das competências básicas de qualquer profissional que lide com computadores. Surge portanto a necessidade da aprendizagem de linguagens de programação para o desenvolvimento desta competência.

O Portugol é um ambiente de desenvolvimento de algoritmos que visa o desenvolvimento do raciocínio algorítmico através de um ambiente simples e com ferramentas visuais para desenvolvimento e depuração dos algoritmos desenvolvidos. Neste ambiente privilegiamos o desenvolvimento de algoritmos ao invés do desenvolvimento de programas.

Utilizamos a linguagem algorítmica, uma pseudo-linguagem de programação que utiliza o português como base das suas instruções.

O Editor de algoritmos tem ferramentas que complementam e corrigem algumas falhas que o aprendiz comete.

O Editor de fluxogramas, é uma abordagem gráfica da programação, que permite o desenvolvimento de algoritmos de forma visual e com ferramentas fazem a tradução de e para linguagem algorítmica.

Qualquer um dos modulos permite fazer a execução do algoritmo e a depuração dos mesmo.
